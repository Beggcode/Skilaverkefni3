Jóla-Spurningakeppnin

Einföld spurningakeppni með jólaþema byggð með React og TypeScript.

Forritið birtir spurningar, leyfir notandanum að velja svar og heldur utan um stig. Þegar allar spurningar eru svaraðar sýnir það lokaeinkunn.

Uppsetning

inn í console
"git clone https://github.com/Beggcode/Skilaverkefni3.git"
"npm install" og svo "npm run dev"
"o + enter" til að ræsa í vafra
